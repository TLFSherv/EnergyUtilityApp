# Energy Utility App
This is the frontend application for the Energy Utility API providing users with information about the API as well as a sign-up/sign-in page for accessing an API token to utilise the API. I'm in the process of developing the app with the aspiration of shipping before the end of Apirl.

## Key Features
- There will be a table listing all the query string parameters and in the table for each of the query string parameter options a checkbox. Checking a checkbox will make a request to the EnergyUtilityAPI with the checked query string parameter option included in the API request. The result will appear next to the table.
- A sign-up and sign-in page with email verification for users to access an API token to utilise the EnergyUtilityAPI in their applications.
- A search box to search the Query String Parameter table by parameter or value.
